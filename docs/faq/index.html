Frequently asked questions

- When I use an endpoint to save some data, where is it stored?#

Your data is stored in a shared database hosted in AWS.

- Can I permanently delete all the data associated with an endpoint?#

Yes - when you delete an endpoint, all the data is deleted.

Remember that the data is not recoverable if you delete it.

- If I use this service for a production application, can I use the same endpoints?#

You should create separate endpoints for production. If something goes wrong during development the production data will be safe.

- How can I contact someone from YAWE?#

Use our email: contact.yawe@gmail.com
